电影天堂iOS客户端
需要配合爬虫使用
所需要环境：
1.因为之前调试爬到的json都只在局域网内有效，所以为了方便已经将爬虫爬到的json存储到了本地（在V2netTool中有说明）
2.本地json请将JSON文件夹下的文件复制到本机Apache服务所指定的文件夹下，并开启apache服务
3.工程中实际使用的json链接为：http://127.0.0.1/latest.json *********
4.因为搜索需要连接java爬虫组员的服务端，所以直接搜索会失败，除非组员在局域网开启tomcat服务（172.24.85.94）
5.虽然你们测试的时候可能无法使用搜索，但是我们小组已经在开发时将搜索功能调试好了，只要局域网环境正常即可实现（搞了一晚上的成果XD）
6.如果在测试时发现其他问题请联系我。。因为我这边调试已经通过了。。
